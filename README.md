## 开发
### 0.安装依赖
docker, nodejs, mysql
### 1.初始化 docker image
```
docker build -f ./paddle/docker/DockerFile ./paddle/docker -t paddle-ocr-lac
```
### 2.安装 node_modules
```
npm install
```
### 3.初始化/重置数据库
```
npm run dev:update-db
```
### 4.本地预览
```
npm run dev
```
